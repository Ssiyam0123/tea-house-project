# Organic Matcha Tea House Landing Page

**Live Demo:** [https://ssiyam0123.github.io/tea-house-project/](https://ssiyam0123.github.io/tea-house-project/)

A premium, elegant, and fully responsive landing page for an organic Japanese Matcha tea brand. Designed with beautiful typography, high-res generated imagery, and structured product storytelling.

## Features
- **Hero Banner**: A stunning large-scale serif header featuring a custom-generated high-resolution image of iced matcha green tea lattes.
- **Origin History**: Describes the history of powdered tea from the Song Dynasty to its integration into Japanese Zen culture by monk Eisai.
- **Production Technology**: Details the step-by-step process of shade-growing, hand-picking, steaming, and slow granite stone-grinding of Tencha leaves.
- **Matcha vs. Others**: Comparative guide highlighting differences between authentic Ceremonial Matcha and generic Green Tea Powder (Fumatsucha).
- **Responsive Layout**: Designed to look majestic and operate fluidly on both desktop screens and mobile viewports.

## Technologies Used
- HTML5
- CSS3 (Vanilla CSS, Responsive Grid & Flexbox)
- Fonts: Google Fonts (Playfair Display & Inter)
- Icons: FontAwesome CDN

## Setup & Running Locally
1. Clone this repository.
2. Open `index.html` in your favorite web browser.
3. (Optional) Run using a local development server like Live Server in VS Code.
